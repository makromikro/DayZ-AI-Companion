# Memory System

## Long-Term Memory

The companion should remember:

- Player name
- Friends
- Enemies
- Bases
- Vehicles
- Important locations
- Important decisions
- Goals

---

## Short-Term Memory

The companion should remember:

- Current conversation
- Recent events
- Temporary plans

---

## World Knowledge

The companion should remember:

- Loot locations
- Vehicle locations
- Base locations
- Dangerous areas
- Safe areas
